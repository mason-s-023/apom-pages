경쟁자 인사이트

🎥
핵심 인사이트
동영상이 킥이다 — 결국 조명빨
경쟁 상품들의 전환 핵심은 동영상 콘텐츠. 회오리·물꽃 분사를 실제로 보여줘야 한다. 단순 상세페이지 이미지로는 경험재의 감각을 전달할 수 없다. 밤 가로등 촬영 → 빛 굴절 + 물꽃 = 조명빨 극대화가 차별화 포인트.

"애가 이거 하나면 엄마를 안 찾아요"

시각적으로 압도하는 분사 성능. 장점 리뷰 1·3위를 동시에 충족. 동영상 한 컷으로 전달 가능.
🌙
밤엔 엄마 인생샷
가로등 + 물꽃 = 조명빨 극대화. SNS 바이럴 트리거. 상징재 15% + 경험재 확장.

Story 흐름 — PASBRO 구조

P
훅 (Hook) — 첫 3초 유입 결정
발견 경로 + 비주얼 + 카피
저번에 워터파크 갔는데 친구들이 이걸로 너무 재밋게 노는거에요! 핫한 아이템이구나 싶어서 이번 여름 제대로 즐기기위해 준비했답니다.
촤라락 엄청 화려하고 재미나네요ㅎ — 아이들은 물꽃놀이라고 이름붙였네요~~
A
공감 (Agitate) — 페인 공감 + 작년 실패
내구 실패 → 종결 전환
별로임 잘부서짐. 하루썻는데 옆에 갈라져서못씀. 택포8천원 버린꼴.
집에 물총 많이 쌓아놓고 안쓰시고 계신 아이들집 있으면 이걸로 종결입니다 ㅋㅋㅋ
S
증폭 (Amplify) — 감정 극대화
엄마 쉼 + 아이 반응
아이 집중도 높고, 에너지 쭉쭉 빠져서 낮잠까지 푹! 풀빌라에서 식사준비 분주한데 아이가 가지고 노느라 엄마를 덜 찾아서 너무너무 만족.
이 제품아니면 안씻겠다고 협박하심ㅋㅋㅋㅋㅋ 물놀이하자고 샤워 매일 즐겁게 하고있습니다
B
전환 (Bridge) — 우리 상품 등장
비주얼 + 기능 + 안전
일반 물총과는 다른 분사 방식으로 정말 불꽃 아니, 물꽃놀이, 워터밤 분위기가 살아나요. 건전지 없이 물만 채워서 바로 쓸 수 있고, 수동식이라 아파트 물놀이장·놀이터에서도 사용 가능.
아파트 물놀이장에서 물총은 사용 못하게 하는데 이건 상관 안하더라구요 / KC 인증까지 완료돼서 안전성도 OK
R
가치 (Roi) — 경험재 최대화
엄마 쉼 + 아이 인싸 + 사회적 공유
물있는곳에서 진짜 엄빠들은 쉴수있다는 커다란장점. 수영장 가서 주변에 어디서 샀냐 물어봐서 여럿 소개했네여. 밤엔 가로등 아래서 물을 발사하면 폭죽처럼 예쁜 인생샷도.
욕조에 넣어주면 목욕이 놀이시간으로 변신! 분수 뿜뿜~ 회오리 슝슝~ 목욕 거부하던 아이도 이젠 먼저 들어가요 / 수영장에서 인싸 수준이였어요 다른 물총은 처다도 안보더라구요
O
증거 (cOnfidence) — 신뢰 증명
내구 반박 + 안전 + 별점 스니펫
물놀이마다 챙겨갔는데 둘 다 부서진 곳없이 잘 놀고 가지고왔어요. 내년여름에도 유용히 써먹어. 몇 번 떨어뜨려도 금방 흠집 나거나 깨지는 일 없이 잘 버텨주더라고요.
잘 안망가지고 아이들이 너무좋아합니다 / 생각보다 잘 망가져요 ㅎㅎ …그래도 하루 신나게 뽀사지게 잘 놀았어요~~ (★3도 인정한 경험재)
→
제안 (Offer) — 구매 결정 트리거
가격 + 시간재 + 로켓
이 가격이면 지금 사야 여름에 쓴다 2개 세트 로켓 다음날 도착
솔직히 이 정도 분사력에 이 가격이면 만족스러웠어요! 너무 가격이 좋아서 반신반의로 2개만 시켰는데 10개시켜서 나눠가질걸그랬어요. 긴박감(시간재) + 가격 장벽 제거 + 배송 신뢰로 마지막 망설임을 해소한다.
